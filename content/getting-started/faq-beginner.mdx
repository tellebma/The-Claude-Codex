Vous avez des questions ? C'est normal. Voici les réponses honnêtes aux interrogations les plus courantes, sans langue de bois.

## Est-ce que ça peut casser mon ordinateur ?

**Non.** Claude Code est un outil logiciel comme n'importe quel autre. Il ne touche pas à vos fichiers système, ne modifie pas votre matériel, et ne fait rien sans votre accord explicite.

Plus précisément : Claude Code vous montre toujours ce qu'il va faire *avant* de le faire, et vous demande votre permission. Vous pouvez refuser à tout moment en tapant `n` ou en appuyant sur `Ctrl+C`.

<Callout type="tip" title="Le pire qui puisse arriver">
Si vous faites une erreur de manipulation dans un projet, vous pourrez toujours revenir en arrière avec Git (un système de sauvegarde de versions). Claude Code lui-même peut vous aider à "défaire" ses propres modifications si quelque chose ne vous convient pas.
</Callout>

## Est-ce que l'IA voit mes fichiers personnels : photos, documents, mails ?

**Non.** Claude Code n'accède qu'au dossier dans lequel vous le lancez, et seulement aux fichiers que vous lui montrez explicitement dans la conversation.

Vos photos de famille, vos documents personnels, vos mails, vos mots de passe enregistrés dans le navigateur, tout ça est invisible pour Claude Code. Il n'y a aucune connexion entre votre navigateur, votre espace de stockage cloud, ou vos applications personnelles.

La seule exception : si vous utilisez des MCP (des connecteurs optionnels) qui donnent accès à des outils spécifiques comme Gmail ou Google Drive. Mais ces connexions doivent être configurées explicitement par vous, elles n'existent pas par défaut.

## C'est quoi la différence entre ChatGPT et Claude Code ?

Bonne question. Les deux sont des intelligences artificielles, mais leur usage est très différent.

| | ChatGPT / Claude.ai | Claude Code |
|---|---|---|
| **Interface** | Site web (chat dans le navigateur) | Terminal (ligne de commande) |
| **Ce qu'il fait** | Répond à vos questions, rédige du texte, explique des concepts | Lit, crée et modifie des fichiers sur votre ordinateur |
| **Accès à vos fichiers** | Non (vous copiez-collez du texte) | Oui (dans le dossier courant) |
| **Idéal pour** | Recherche, rédaction, conversation | Développement, automatisation, gestion de fichiers |
| **Coût** | Gratuit ou abonnement séparé | Abonnement Claude Max ou clé API |

En résumé : ChatGPT et Claude.ai sont des chatbots web. Claude Code est un assistant qui travaille directement dans vos fichiers.

## Est-ce que c'est légal ?

**Oui, tout à fait légal.** Claude Code est un produit commercial d'Anthropic, une entreprise américaine légalement constituée. Utiliser Claude Code, c'est comme utiliser un abonnement à n'importe quel autre logiciel SaaS.

Les seules limites légales qui s'appliquent sont les mêmes que pour tout outil : ne pas l'utiliser pour créer du contenu illégal, du spam, ou pour contourner des systèmes de sécurité.

## J'ai besoin d'Internet pour l'utiliser ?

**Oui, une connexion Internet est nécessaire.** Claude Code envoie vos demandes aux serveurs d'Anthropic et reçoit les réponses. Sans connexion, l'outil ne peut pas fonctionner.

En revanche, les fichiers créés ou modifiés restent sur votre ordinateur, même hors connexion.

## Ça marche sur Chromebook / tablette / téléphone ?

**Non, du moins pas directement.** Claude Code nécessite un vrai système d'exploitation avec un terminal (macOS, Windows, Linux). Les Chromebooks en mode standard, les tablettes et les téléphones ne supportent pas ce type d'installation.

Si vous avez un Chromebook, il existe une solution via "Linux (bêta)" dans les paramètres, mais c'est assez technique et pas recommandé pour les débutants.

Pour commencer confortablement, il vous faut :
- Un Mac (n'importe quel modèle des 5 dernières années)
- Un PC Windows 10 ou 11
- Un ordinateur Linux

## Est-ce que je peux annuler à tout moment ?

**Oui, sans engagement.** L'abonnement Claude Max est mensuel et se gère depuis votre espace Anthropic. Vous pouvez annuler quand vous voulez, sans frais de résiliation, sans délai de préavis.

Si vous utilisez la clé API (pay-as-you-go), il n'y a pas d'abonnement du tout, vous payez uniquement ce que vous consommez.

## C'est dangereux pour mon emploi ?

**Non, c'est exactement l'inverse.** Claude Code est un outil d'augmentation, pas de remplacement.

Pensez à la calculatrice : elle n'a pas supprimé les comptables, elle leur a permis de travailler plus vite et de se concentrer sur des tâches à plus haute valeur ajoutée. Claude Code fait pareil pour les tâches intellectuelles.

Les personnes qui maîtrisent ces outils seront plus productives, plus créatives et plus compétitives sur le marché du travail, pas l'inverse.

## Faut-il savoir coder pour utiliser Claude Code ?

**Non, ce n'est pas obligatoire.** Claude Code peut vous aider avec des tâches non-techniques : rédiger des emails, analyser des documents, créer des présentations, organiser des données dans des fichiers...

Cela dit, plus vous savez ce que vous voulez et plus vous êtes capable de vérifier le résultat, mieux ce sera. Quelques notions de base (comme comprendre ce qu'est un fichier HTML ou une feuille de calcul CSV) vous aideront à mieux profiter de l'outil.

## Claude Code va-t-il tout faire à ma place ?

**Non, et c'est une bonne chose.** Claude Code est un assistant, pas un robot autonome. Il fait ce que vous lui demandez, mais vous restez le décideur.

En pratique, vous devrez :
- Formuler des demandes claires
- Vérifier les résultats
- Donner votre accord avant chaque action importante
- Corriger et ajuster quand le résultat ne correspond pas à ce que vous vouliez

C'est un aller-retour constant. La qualité de ce dialogue est ce qui fait la différence entre une expérience frustrante et une expérience productive.

## Et si Claude Code fait une erreur ?

**C'est prévu, et gérable.** Claude Code fait parfois des erreurs. Il peut mal comprendre votre demande, oublier un détail, ou proposer une solution incorrecte.

Voici comment réagir :
1. **Signalez l'erreur** : dites simplement "ce n'est pas ce que je voulais, voici ce qui ne va pas..."
2. **Soyez précis** : plus vous expliquez ce qui ne va pas, mieux il pourra corriger
3. **Annulez si nécessaire** : vous pouvez toujours revenir en arrière avec Git, ou demander à Claude Code de défaire ses modifications

## Mes données sont-elles privées ?

Anthropic collecte les conversations pour améliorer ses modèles, mais vous pouvez configurer votre compte pour limiter cela. Les données sont stockées sur des serveurs américains.

**À ne jamais partager avec Claude Code** : mots de passe, numéros de carte bancaire, informations médicales confidentielles, données personnelles sensibles de tiers.

**Consultez** la [politique de confidentialité d'Anthropic](https://www.anthropic.com/privacy) pour les détails complets.
