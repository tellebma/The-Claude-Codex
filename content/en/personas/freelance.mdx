## Your profile

You work solo or with a small team. Your time is your most precious resource: every hour saved is a billable hour (or an hour of rest). You're looking for concrete productivity gains, not theory.

This path gets straight to the point. Quick installation, optimal configuration, then the tools and workflows that actually save time.

<Callout type="info" title="Estimated time">
Plan 2 hours for the essential steps (1 through 6). Steps 7 and 8 are bonuses to go further.
</Callout>

## Your journey

<Steps>
<Step title="Install in 5 minutes" stepNumber={1}>

No time to waste: install Claude Code and set up your environment as fast as possible.

**Read**: [Prerequisites and installation](/getting-started/installation), then [Environment setup](/getting-started/environment-setup)

</Step>

<Step title="Generate your custom configuration" stepNumber={2}>

The interactive configurator generates a CLAUDE.md, a settings.json, and an MCP configuration tailored to your stack and habits. It's the fastest way to properly set up Claude Code.

**Read**: [Interactive configurator](/configurator)

</Step>

<Step title="Master effective prompting" stepNumber={3}>

A good prompt saves you minutes on every interaction. Learn the basics, then the directives that make the difference.

**Read**: [Prompting basics](/prompting/basics), then [Advanced directives](/prompting/directives)

</Step>

<Step title="Install productivity MCPs" stepNumber={4}>

MCPs connect Claude Code to your everyday tools. Gmail, Slack, GitHub, databases: pick the ones that match your workflow.

**Read**: [Understanding MCPs](/mcp/what-are-mcps), then [Top productivity MCPs](/mcp/best-productivity), then [Top development MCPs](/mcp/best-development)

</Step>

<Step title="Add essential plugins" stepNumber={5}>

Plugins enrich Claude Code with specialized features. Start with the essentials, then explore those that match your trade.

**Read**: [Understanding plugins](/plugins/what-are-plugins), then [Top essential plugins](/plugins/best-essential)

</Step>

<Step title="Automate with Skills" stepNumber={6}>

Skills turn repetitive workflows into reusable commands. Code review, test generation, documentation: create your own recipes.

**Read**: [What is a Skill?](/skills/what-are-skills), then [Create a custom Skill](/skills/create-custom)

</Step>

<Step title="Optimize your costs" stepNumber={7}>

As a freelancer, every dollar counts. Understand the billing system, compare plans, and apply optimization strategies.

**Read**: [Real costs of Claude Code](/content/real-costs-claude-code)

</Step>

<Step title="Build complete MCP workflows" stepNumber={8}>

Combine multiple MCPs into an automated workflow. For example: receive a GitHub ticket, analyze the code with Context7, implement the fix, and open a PR, all in a single command.

**Read**: [First MCP workflow](/mcp/first-workflow), then [Prompt chaining and orchestration](/prompting/chaining-and-agents)

</Step>
</Steps>
