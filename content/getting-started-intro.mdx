## Qu'est-ce que Claude Code ?

Claude Code est un **assistant IA en ligne de commande** developpe par Anthropic. Il vit dans votre terminal et comprend votre contexte de travail : fichiers, arborescence, historique Git, et bien plus encore.

Contrairement a un chatbot classique, Claude Code **agit** dans votre environnement reel. Il peut creer des fichiers, executer des commandes, analyser votre code et vous accompagner dans vos projets de A a Z.

<Callout type="tip" title="Pour qui est ce guide ?">
Ce guide s'adresse aussi bien aux developpeurs experimentes qu'aux debutants complets. Chaque concept est explique avec des analogies simples et des exemples concrets.
</Callout>

## Pourquoi utiliser Claude Code ?

Voici les principaux avantages :

- **Gain de temps** : automatisez les taches repetitives et accelerez votre workflow
- **Apprentissage** : Claude explique chaque etape et vous aide a comprendre le code
- **Qualite** : code revue, tests automatises et bonnes pratiques integrees
- **Accessibilite** : pas besoin d'etre developpeur pour creer des projets concrets

## Les trois piliers de Claude Code

<Card title="1. Le Terminal" variant="accent">
Claude Code fonctionne dans votre terminal. C'est son environnement naturel, la ou il est le plus puissant. Il peut lire vos fichiers, executer des commandes et interagir avec votre systeme.
</Card>

<Card title="2. Les MCP (Model Context Protocol)" variant="highlight">
Les MCP sont des connecteurs qui etendent les capacites de Claude Code. Connectez-le a GitHub, Slack, Gmail, vos bases de donnees et des dizaines d'autres outils.
</Card>

<Card title="3. Les Skills" variant="default">
Les Skills sont des recettes reutilisables qui guident Claude Code pour executer des taches complexes de maniere reproductible. Creez les votres ou utilisez ceux de la communaute.
</Card>
