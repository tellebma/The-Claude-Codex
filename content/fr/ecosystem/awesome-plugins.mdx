## TL;DR

- Les **plugins** sont des intégrations packagées distribuées via marketplace. Plus structurés qu'un skill, plus légers qu'un agent custom.
- Référence officielle : [`anthropics/claude-plugins-official`](https://github.com/anthropics/claude-plugins-official) (19k★) — directory managé par Anthropic, plugins validés.
- 3 plugins community valent l'installation immédiate : **claude-hud** (HUD visuel temps réel), **compound-engineering-plugin** (workflows cross-tool), **claudian** (intégration Obsidian).
- 2 marketplaces alternatives ouvrent au-delà du repo officiel : `alirezarezvani/claude-skills` (245+ plugins) et `jeremylongshore/claude-code-plugins-plus-skills` (425 plugins).

<Callout type="info" title="Plugin vs Skill vs Agent">
Un **plugin** se distribue via marketplace, expose une API publique, et a un cycle de vie versionné. Un **skill** est un fichier markdown isolé. Un **agent** est une délégation de tâche. Voir [`/skills/comparison`](/skills/comparison) pour la matrice.
</Callout>

## Comment choisir

Trois étapes :

1. **Commencer par l'officiel** : parcourir [`anthropics/claude-plugins-official`](https://github.com/anthropics/claude-plugins-official) en premier. C'est la base validée. Tu y trouveras les plugins essentiels pour le quotidien (formattage, lint, tests).
2. **Ajouter 1-2 plugins community** : `claude-hud` est un must-have visuel (savoir ce qui se passe pendant les longs runs). `compound-engineering-plugin` si tu jongles entre plusieurs outils IA.
3. **Marketplaces tierces** : seulement si tu cherches un plugin précis non trouvé ailleurs. Faire attention à la sécurité (plugins non audités peuvent exécuter du code arbitraire).

<Callout type="warning" title="Sécurité des plugins tiers">
Un plugin a accès à ton CLI et potentiellement à ton filesystem. Avant d'installer un plugin community, lis le code source (ou au minimum le README + le manifest). Privilégie les plugins ayant > 10k★ et un mainteneur actif. Les plugins « tonsofskills.com » et autres marketplaces non officielles ne sont pas audités par Anthropic.
</Callout>
