## Resumen

- Los **plugins** son integraciones empaquetadas que se distribuyen mediante marketplace. Más estructurados que un skill, más ligeros que un agente personalizado.
- Referencia oficial: [`anthropics/claude-plugins-official`](https://github.com/anthropics/claude-plugins-official) (19k★), directorio gestionado por Anthropic con plugins validados.
- 3 plugins de la comunidad merecen instalación inmediata: **claude-hud** (HUD visual en tiempo real), **compound-engineering-plugin** (flujos de trabajo cross-tool), **claudian** (integración con Obsidian).
- 2 marketplaces alternativos amplían el alcance más allá del repositorio oficial: `alirezarezvani/claude-skills` (245+ plugins) y `jeremylongshore/claude-code-plugins-plus-skills` (425 plugins).

<Callout type="info" title="Plugin vs Skill vs Agente">
Un **plugin** se distribuye mediante marketplace, expone una API pública y tiene un ciclo de vida versionado. Un **skill** es un archivo markdown aislado. Un **agente** es una delegación de tarea. Consulta [`/skills/comparison`](/skills/comparison) para ver la matriz comparativa.
</Callout>

## Cómo elegir

Tres pasos:

1. **Empieza por lo oficial**: explora primero [`anthropics/claude-plugins-official`](https://github.com/anthropics/claude-plugins-official). Es la base validada. Ahí encontrarás los plugins esenciales para el día a día (formateo, lint, tests).
2. **Añade 1-2 plugins de la comunidad**: `claude-hud` es imprescindible a nivel visual (saber qué está pasando durante las ejecuciones largas). `compound-engineering-plugin` si alternas entre varias herramientas de IA.
3. **Marketplaces de terceros**: solo si buscas un plugin concreto que no encuentras en otro sitio. Presta atención a la seguridad (los plugins no auditados pueden ejecutar código arbitrario).

<Callout type="warning" title="Seguridad de los plugins de terceros">
Un plugin tiene acceso a tu CLI y potencialmente a tu sistema de archivos. Antes de instalar un plugin de la comunidad, lee el código fuente (o como mínimo el README y el manifest). Prioriza los plugins con más de 10k★ y un mantenedor activo. Los plugins de "tonsofskills.com" y otros marketplaces no oficiales no están auditados por Anthropic.
</Callout>
